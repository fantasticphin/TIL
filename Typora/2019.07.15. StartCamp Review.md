# git & GitHub

**git** = 분산 버전 관리 시스템, 파일별 수정된 사항만 스냅샷을 찍어, 특정 과거 파일로 쉽게 돌아갈 수 있다.

매우 적은 메모리가 장점

1. 작업(수정)파일
2. 커밋할 목록
3. 쌓인 커밋들
4. 깃헙 업로드

**GitHub** = 원격저장소 / **GitHub Pages** : 수레바퀴를 두 번 만들지 말고, 공유된 자료를 가져올 수 있음
